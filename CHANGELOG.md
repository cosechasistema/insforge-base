# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2026-03-08

### Added
- Initial template setup with Nuxt 4 + Vuetify 3 + InsForge SDK
- Authentication system (login, register, verify-email) with role-based access
- Layout system (sidebar, toolbar, offline banner)
- Items CRUD as example module (list, detail, create, edit, delete)
- Report utilities (PDF/Excel export)
- Unit and E2E testing infrastructure
- Skills: crud-search, crud-report, changelog
- PWA support with offline caching
